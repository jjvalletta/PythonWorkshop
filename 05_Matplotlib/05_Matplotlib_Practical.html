import numpy as np
import matplotlib.pyplot as plt

# Generate sinusoidal data

plt.xlabel("Time")
plt.ylabel("Amplitude")
plt.title("My first Python plot")

3D plots¶

Let us generate some artificial 3D data. In this case we will plot the exponential function

$$ z = e^{-x^2 - y^2} $$

First we need to create a grid/mesh, a 2D Numpy array, on which $z$ is computed on.

Now we have three variables ($x, y ,z$) represented as a 2D Numpy matrix. To plot in 3D, we first set the figure's axis to 3D then plot the data.

In [9]:
from mpl_toolkits.mplot3d import Axes3D
from matplotlib import cm

# Create figure amenable for 3D plotting
hFig = plt.figure() # hFig - a handle to the figure object
hAx = hFig.gca(projection="3d") # hAx - a handle to the axis object

# Plot the surface
hSurf = hAx.plot_surface(x, y, z, cmap=cm.inferno) # hSurf - a handle to the 3D surface object
plt.colorbar(hSurf, shrink=.5)
