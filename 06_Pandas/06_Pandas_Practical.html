Introduction¶

Pandas is Python's data analysis toolkit, used for:

  • Reading/writing data of different formats (e.g CSV, SQL database)
  • Creating and manipulating data frames (akin to R)
  • Handling missing data
  • Meging/joining data sets
  • Reshaping/pivoting data sets
  • Time-series analysis
  • $\ldots$

A very useful Pandas cheat sheet can be found here

Creating a data frame¶

This can be created by specifying a dictionary where each entry corresponds to a column in the dataset. In this case let us consider collecting data from various individuals (samples) at times t0, t1,...,t5

In [2]:
import pandas as pd

df = pd.DataFrame(
        {"Sample" : ["R100" , "R201", "R203", "R340", "R453"],
         "t0" : [0.2, 0.1, 0.3, 0.25, 0.13], 
         "t1" : [1.3, 1.8, 0.8, 1.5, 0.6],
         "t2" : [2.8, 3.1, 1.9, 2.3, 1.8],
         "t3" : [3.2, 3.7, 2.3, 3.5, 2.5],
         "t4" : [1.2, 1.8, 3.9, 1.3, 3.7],
         "t5" : [0.7, 0.4, 3.4, 0.3, 3.6]})
